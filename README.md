## What is it

It’s a full-stack note-taking web application where users can create, read, update, and delete notes.

I built it to practice end-to-end web development and understand how frontend, backend, and database layers work together.

## Tech Stack
The frontend was built with React and Tailwind CSS, the backend with Node.js and Express, and MongoDB was used as the database. I also implemented rate limiting with Upstash Redis.

I implemented the RESTful API, connected it to MongoDB, built the frontend pages, handled error states, and deployed the application with Render.

## Learning
The biggest learning was understanding how data flows from the frontend to the backend and then to the database. I also learned how to structure API endpoints and protect the backend with rate limiting.
